# UniCredit Mobile Bank

A mobile banking application built with React Native and Expo.

## 🌐 Live Demo

The web version of the app is deployed and accessible at:

**👉 https://dj681.github.io/unicredit-mobile-bank/**

> **Note:** To enable GitHub Pages, go to the repository **Settings → Pages**, set the source to **"Deploy from a branch"**, select the **`gh-pages`** branch and **`/ (root)`** folder, then click **Save**.

## Development

```bash
# Install dependencies
npm install

# Start the development server
npm start

# Build the web version
npm run build:web
```

## Deployment

The app is automatically deployed to GitHub Pages via GitHub Actions on every push to `main`.
